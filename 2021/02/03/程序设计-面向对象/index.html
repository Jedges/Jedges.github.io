面向对象与过程

面向对象程序设计

  • 一个面向对象程序由一些对象构成
  • 对象是由一些数据及可施于这些数据上的操作所组成的封装体，
  • 对数据的操作是通过向包含数据的对象发送消息（调用对象提供的操作）来实现。
  • 对象的特征由相应的类来描述，一个类的特征可以从其它的类继承。
  • 面向对象程序的执行过程体现为各个对象之间相互发送和处理消息。

数据抽象与封装。

  1. 数据抽象

数据的使用者只需要知道对数据能实施那些操作以及这些 操作之间的关系，而不必知道数据的具体表示形式。

  1. 封装

把数据及其操作作为一个整体来进行实现。（封装）
数据的具体表示对使用者是不可见的，对数据的访问（使用）只能通过封装体所提供的对外接口（允许的操作）来完成。

  1. 数据抽象与封装是面向对象程序设计的基础
  • 实例：栈数据的实现

类

  1. 对象构成了面向对象程序设计的·1基本计算单位，而对象的特征则有相应的类来描述
  2. 定义： class <类名> { <成员描述> } ;
    • 成员包括：数据成员和成员函数。
    • 类成员标识符的作用域为整个类定义范围。
  3. 类成员的访问控制(默认访问控制为private)
    • private：不受限制。
    • public：只能在本类和友元的代码中访问。
    • protected: 只能在本类、派生类和友元的代码中访问。

成员函数

  1. 成员函数是对类成员所能实施的操作描述。
  2. 成员函数的实现可以放在类定义中。也可以放在类定义外。但需要进行声明。并且定义是需要用类名受限，区别于全局函数。
  3. 成员函数可以重载。遵循一般规则。

数据成员

  • 类的数据成员可以是c++任何类型。但是，在声明一个数据或成员的类型时，如果未见到相应类型的定义或定义未完成时，则该成员的类型一般只是这些类型的引用和指针类型。

对象

类属于类型范畴的程序实体，它一般存在于静态（运行前）的程序中。而动态（运行中）的面向对象程序则是由对象构成。对象在程序运行时创建。

对象的操作：
  • 通过调用类的成员函数对其对象a进行操作；
    • 在类的外部，通过对象来访问类的成员时要受到类成员访问控制的限制。
  • 同类对象之间可以互相赋值。
  • 可以把对象作为实参传给函数。
  • 把对象作为函数的返回值。
构造函数
  1. 在使用对象前，需要对对象储存空间中的数据进行初始化;
  2. 构造函数： 名字与类名相同，无返回值类型。创建对象时，构造函数会自动调用来初始化对象的数据成员；
  3. 构造函数可以重载。
    • 创建对象时，可显示指定调用某个构造函数进行初始化。
    • 没有指定调用某种函数，则调用默认构造函数。
    • 一旦定义某种构造函数，则编译器不会在提供默认构造函数。
  4. 对于常量和引用数据成员，不能在说明它们时初始化，也不能采用赋值操作在构造函数中对它们初始化。
  5. 成员初始化表：可以在构造函数的函数头和函数体之间加入一个成员初始化表来对常量和引用数据成员进行初始化。
    • 当类中有常量数据成员和引用数据成员时，如果没有定义构造函数，编译器不会生成默认构造函数，因为无法对常量和引用类型进行初始化。所以一定要在成员初始化表中对他们进行初始化。
对象的消亡前处理————析构函数
  • 析构函数，它的名字为“~<类名>”，没有返回类型、不带参数、不能被重载。
  • 一个对象消亡时，系统在收回它的内存空间之前，将会自动调用析构函数。
  • 可以在析构函数中完成对象被删除前的一些清理工作（如：归还对象额外申请的内存空间）（比如对象自己申请的动态变量）。
拷贝构造函数
  • 在创建一个对象时，若用一个对象本类对其进行初始化，则需要拷贝构造函数。
  • 拷贝函数的参数类型为本类的引用
  • 拷贝构造函数的参数类型为本类的引用。例如：
  • 隐式拷贝构造函数。
  • 当初始化时含有指针时，隐式构造函数会将两个对象的指针指向同一块内存区域。并且在其中一个消亡后，会使得另一个也无法正常使用，所以需要自己定义新的拷贝构造函数。
this指针
  • 每一个成员函数都有一个隐藏的形参this。通常情况下可以省略不写，但如果成员函数中要把this所指向的对象作为一个整体进行操作，则必须显示的使用this指针。
常成员函数

1.定义：

  1. 表示该函数只改变数据成员的状态，而不修改数据成员的值。
    • 但是可以改变指针所指向的对象的值。
  2. 对常量对象进行调用非const成员函数非法。

操作符重载

  1. 部分操作符不可以进行重载，比如:“.(成员选择符)”，“::(域解析符)”，“?:(条件操作符)”，“sizeof(数据占内存大小)”，“.*(间接成员选择符)”;
  2. 类默认提供赋值操作符重载。
