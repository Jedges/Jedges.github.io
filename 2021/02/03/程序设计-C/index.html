访问动态变量

动态变量

  • 在程序运行中，根据需要创建的变量。
  • 生成 new (type) // (type *)malloc(sizeof(type);
  • 动态变量需要显示的撤销 delete p //delete [ ]p // free§
    • new创建的动态变量用delete撤销，malloc创建的用free来撤销*
    • 撤销动态数组时，指针变量必须指向数组的第一个元素
    • 悬浮指针：撤销动态变量后，指针指向一个无效空间，而不是默认为0；
  • 动态变量存储于堆空间中，动态变量没有名字，只能通过指针进行间接访问

动态数组

1.数组的大小是一个输入的变量。

2.数组的大小完全不知道，取决于输入数的多少。先申请较小的空间，根据情况进行重新申请。
